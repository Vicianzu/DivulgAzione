## 🛠️ Tecnologie

- **HTML5 Canvas** — grafica 2D animata
- **JavaScript ES6+** — logica delle simulazioni, gestione degli stati
- **Animazioni `requestAnimationFrame`** — rendering fluido senza dipendenze
- **SVG** — visualizzazioni geometriche vettoriali

Nessuna dipendenza esterna: ogni file è autonomo e si apre direttamente nel browser.
